## 📦 Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/Enhle-debug/minenhle-portfolio.git

# Navigate into the project
cd minenhle-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
